# personal-tboc

Tiny Bits of Carbon — blog/website built with Jekyll and hosted on GitHub Pages.

## Stack

- Jekyll (Ruby)
- GitHub Pages
- SASS for styling

## Structure

- `_posts/` — blog posts
- `_pages/` — static pages
- `_layouts/` — page layouts
- `_includes/` — reusable HTML partials
- `_sass/` — SASS stylesheets
- `_data/` — data files
- `_config.yml` — Jekyll configuration
- `assets/` — static assets

## Branch

- Main branch is `master` (deployed via GitHub Pages)
